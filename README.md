Virasat-Namma Guide – (Tourism & Travel)   
Android Application with GenAI Integration    
Pooja Manjunath Waingankar   
1. Introduction   
Virasat-Namma is an Android app to explore hidden heritage sites in Karnataka. It highlights temples, 
monuments, and cultural places. Many of these sites are not listed on popular platforms. This reduces awareness 
of their importance.  
The app uses Generative AI to provide useful information and insights. It includes QR scanning and a digital 
check-in feature. Users can explore nearby locations easily. It helps promote cultural awareness and tourism. 
2. Problem Statement   
Tourists face difficulty in finding lesser-known heritage sites. Many places are not listed on apps like Google 
Maps. Information about them is limited or unclear. This makes exploration difficult for users. 
Due to this, many heritage sites remain unnoticed. Awareness about local history is reduced. There is no smart 
system for personalized guidance. This limits the overall tourism experience.  
3. Objectives   
• To develop an Android app for heritage exploration  
• To identify and showcase hidden heritage sites 
• To provide accurate historical information   
• To use GenAI for smart recommendations 
• To enable location-based discovery 
• To include QR-based information access 
• To promote cultural awareness and tourism 
4. Scope   
In Scope   
• User registration and login 
• Location-based heritage site discovery  
• Display of nearby historical places 
• GenAI-based information and suggestions 
• QR code scanning for site details 
• Digital check-in for visited places 
• Multi-language support 
• Simple and user-friendly interface 
5. Functional Requirements   
1. The system shall allow user registration and login   
2. The system shall detect user location using GPS    
3. The system shall display nearby heritage sites  
4. The system shall provide detailed site information 
5. The system shall generate GenAI-based suggestions    
6. The system shall allow QR code scanning 
7. The system shall enable digital check-in for visits 
8. The system shall support multiple languages    
9. The system shall provide search and filter options 
6. Non-Functional Requirements   
• Performance: The app should load nearby heritage sites and details within 2–3 seconds 
• Reliability: The system should work smoothly without crashes and store user data properly   
• Security: User data and location information must be protected 
• Usability: The interface should be simple and easy for all users 
• Scalability: The app should handle increasing users and data 
• Compatibility: The app should support Android version 9.0 and above 
7. User Requirements   
Tourist (User) Requirements:   
• Easy discovery of nearby heritage sites  
• Access to accurate historical information  
• Personalized recommendations based on interests  
• Ability to check-in and track visited places   
Vendor (Local Provider) Requirements:   
• Ability to add and manage heritage site details 
• Visibility to tourists   
• Simple and easy system without technical complexity 
8. System Requirements   
Hardware Requirements:   
• Android smartphone with GPS and camera support  
Software Requirements:   
• Android OS (version 9 or above)   
• Internet connection for real-time data   
• Google Maps API for location services  
• Android Studio for development 
• GenAI API integration   
9. Data Requirements   
Location Table:   
• Location ID (Primary Key)   
• Site Name 
• Category (Temple / Monument / Heritage Site)  
• Description   
• Latitude & Longitude   
• Historical Details 
• QR Code Data 
• Images   
User Table:   
• User ID   
• Name   
• Email   
• Preferences   
Review Table:   
• Review ID   
• User ID (Foreign Key)   
• Location ID (Foreign Key)   
• Review Content (Text/Voice/Image)   
• Rating   
10. Constraints   
• Limited development time (6 weeks)   
• Use of free or limited APIs 
• Accuracy depends on available data 
• Internet connectivity required for real-time features   
11. Expected Outcome   
The Virasat-Namma app will provide an easy way for users to discover and explore hidden heritage sites in 
Karnataka. It will improve awareness of cultural and historical places while offering a simple and interactive 
user experience. The app will also encourage tourism and help preserve local heritage through digital access.
